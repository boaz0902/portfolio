## 🎯 30-MINUTE ACTION PLAN

### Step 1: Personalize (10 minutes)
```bash
Edit index.html:
1. Replace "Obed" with your name
2. Update your title/role
3. Add your social links (GitHub, LinkedIn)
4. Update your contact email
```

### Step 2: Configure Email (10 minutes)
```bash
1. Go to emailjs.com (free signup)
2. Create email service + template
3. Copy credentials:
   - Public Key → index.html line 12
   - Service ID → assets/js/contact.js
   - Template ID → assets/js/contact.js
```

### Step 3: Deploy (10 minutes)
```bash
1. Push to GitHub
2. Import to Vercel
3. Add environment variables
4. Your site is live!
```
