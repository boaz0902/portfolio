## 🚀 QUICK START (30 Minutes)

### Step 1: Update Your Information (5 min)
```bash
# Edit index.html and replace:
- "Obed" → Your name (throughout file)
- GitHub link → Your GitHub profile
- LinkedIn link → Your LinkedIn profile
- your-email@example.com → Your email
```

### Step 2: Set Up EmailJS (10 min)
```bash
# 1. Go to emailjs.com and sign up
# 2. Create email service and template
# 3. Copy credentials to:
#    - index.html line 12: PUBLIC_KEY
#    - assets/js/contact.js lines 95-99: SERVICE_ID, TEMPLATE_ID, email
```

### Step 3: Test Locally (5 min)
```bash
# Open index.html in browser
# Test: Scroll, click links, try contact form, check mobile view
```

### Step 4: Deploy (10 min)
```bash
# Push to GitHub
git add .
git commit -m "Initial portfolio"
git push origin main

# Then import to Vercel dashboard
# Your site will be live instantly!
```
