## 🔧 Essential Commands Reference

```bash
# Install dependencies
npm install

# Build Tailwind CSS
npm run build

# Watch Tailwind (auto-build on save)
npm run dev

# Deploy to Vercel
vercel --prod

# Check for updates
npm outdated
```

## 📂 Adding New Files

### Add Project Image
```
1. Create optimized image (< 500KB)
2. Save to: assets/images/project-name.jpg
3. Update HTML: <img src="assets/images/project-name.jpg">
```

### Add Custom CSS
```
1. Add to: assets/css/styles.css
2. Create new class:
   .custom-class { /* styles */ }
3. Use in HTML: class="custom-class"
```

### Add JavaScript Function
```
1. Add to: assets/js/main.js (or new file)
2. Reference in: index.html
3. Test in browser console
```

## 💾 Backup & Version Control

### Git Workflow
```bash
git add .                    # Stage changes
git commit -m "message"      # Commit changes
git push origin main         # Push to GitHub
```

### Create Backup
```bash
git clone <repo-url> backup  # Clone to backup folder
git branch                   # List branches
git log                      # View commit history
```

## 🆘 Emergency Reference

**Something broke?** Use Git to revert:
```bash
git status                   # See what changed
git diff                     # See exact changes
git checkout -- .            # Revert all changes
git reset --hard HEAD^       # Undo last commit
```
