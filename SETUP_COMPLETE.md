## 🎯 Next Steps (30 Minutes to Live!)

### Step 1: Update Your Information (5 minutes)
1. Open `index.html` in your editor
2. Replace "Obed" with your name (entire file)
3. Update social links in footer (GitHub, LinkedIn)
4. Save the file

### Step 2: Set Up EmailJS (10 minutes)
1. Visit https://www.emailjs.com
2. Sign up (free account)
3. Follow `EMAILJS_SETUP.md` for credentials
4. Update these in `index.html` and `assets/js/contact.js`:
   - Your EmailJS Public Key
   - Service ID
   - Template ID
   - Your email address

### Step 3: Test Locally (5 minutes)
1. Open `index.html` in your web browser
2. Test all sections: scroll, click, hover
3. Try the contact form
4. Test chatbot
5. Check mobile responsiveness (F12 → Device mode)

### Step 4: Deploy to Vercel (10 minutes)
1. Push to GitHub:
   ```bash
   git add .
   git commit -m "Initial portfolio commit"
   git push origin main
   ```

2. Go to vercel.com → Import project from GitHub
3. Add environment variables
4. Click Deploy
5. Your site is live! 🚀
